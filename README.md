# GelatoMao's Blog

我的个人技术博客，基于 [Astro](https://astro.build/) + [AstroPaper](https://github.com/satnaing/astro-paper) 主题构建。

🔗 **在线访问**: [https://gelatomao.github.io](https://gelatomao.github.io)

## 技术栈

- **框架**: Astro 6
- **主题**: AstroPaper v6
- **样式**: Tailwind CSS 4
- **搜索**: Pagefind
- **部署**: GitHub Pages + GitHub Actions

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview
```

## 写文章

在 `src/content/posts/` 目录下创建 `.md` 或 `.mdx` 文件即可。
